![logo]

## Features
* Clippy
* A pistol and M16 to shoot Clippy
* 3 stages with waves of Clippy


## How to Play
* WASD or Arrow Keys to move
* Move mouse and click to shoot
* Escape to pause
* Q to switch weapons
